The Library Management System (LMS) is a desktop-based application developed using JavaFX that automates the core operations of a library. It allows the librarian to manage books, issue and return records, calculate fines, and track borrower details efficiently. The system also supports user login authentication and real-time status of books.

This project is designed for academic purposes and demonstrates the practical implementation of Object-Oriented Programming (OOP) concepts and JavaFX GUI development.
